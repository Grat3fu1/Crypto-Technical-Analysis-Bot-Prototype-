# Crypto-Technical-Analysis-Bot-Prototype-
A crypto bot that gives signal (Golden cross, death cross, significanty above / below MA, etc...) and other infos using Python

## Features
- Fetches market data via `ccxt`
- Computes moving averages
- Detects MA crossover signals (Golden/Death Cross)
- Prints/flags signals for observation (no trading execution)

## Libraries
- Python
- ccxt
- pandas, numpy
